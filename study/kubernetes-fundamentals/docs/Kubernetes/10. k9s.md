# Kubernetes k9s Terminal UI

## üéØ What is k9s?

**k9s** is a terminal-based UI for interacting with Kubernetes clusters. It provides a vim-like interface with keyboard shortcuts for efficient cluster management and monitoring.

### Key Features
- **Real-time cluster monitoring** - Live updates of cluster state
- **Vim-like navigation** - Familiar keyboard shortcuts
- **Resource exploration** - Browse all Kubernetes resources
- **Quick actions** - Delete, edit, describe resources instantly
- **Multi-cluster support** - Switch between different clusters
- **Built-in search** - Find resources quickly

## üöÄ Pro Tips

### Efficiency Tips
- **Use search liberally** - `/` is your friend for finding resources quickly
- **Learn the shortcuts** - `:deploy`, `:pods`, etc. are faster than navigating menus
- **Sort strategically** - `Shift+S` for status helps identify issues quickly
- **Use describe first** - `Enter` gives you detailed info before taking actions

### Troubleshooting Workflow
```bash
1. :pods          # Start with pods overview
2. Shift+S        # Sort by status to see issues
3. /error         # Filter problematic pods  
4. Enter          # Describe the pod
5. l              # Check logs
6. s              # Shell in if needed
```

### Monitoring Workflow  
```bash
1. :nodes         # Check cluster health
2. :pods          # Monitor pod status
3. Shift+A        # Sort by age to see recent changes
4. :events        # Check cluster events
5. /warning       # Filter for warnings
```

## üéØ Best Practices

### Daily Operations
- **Start with cluster overview** - Check nodes, then pods
- **Use sorting effectively** - Sort by status to catch issues early  
- **Regular event checking** - `:events` shows cluster health
- **Namespace awareness** - Use `Ctrl+a` to toggle all namespaces view

### Troubleshooting
- **Follow the data flow** - Ingress ‚Üí Service ‚Üí Pod ‚Üí Container
- **Check events first** - Often reveals the root cause
- **Use logs extensively** - `l` command is invaluable
- **Describe before acting** - `Enter` gives context before changes

### Performance
- **Filter aggressively** - Use `/` to reduce noise
- **Learn keyboard shortcuts** - Much faster than mouse navigation
- **Use resource-specific views** - More efficient than generic browsing
- **Bookmark common workflows** - Develop muscle memory for frequent tasks
